# Smart Cut

Optimisez vos achats en répartissant les longueurs sur les planches et chevrons du magasin.

Smart Cut est une application web qui calcule comment découper vos pièces à partir des longueurs disponibles pour minimiser le gaspillage et optimiser vos achats de bois.

## Fonctionnalités

- **Produits en stock** — Définissez les longueurs disponibles (ex. 2,4 m, 3 m)
- **Listes de découpe** — Indiquez les longueurs à couper pour chaque produit
- **Kerf** — Optionnel : épaisseur de coupe pour des résultats plus précis
- **Sauvegarde** — Enregistrez et chargez vos projets en JSON

## Démarrage

```bash
bun install
bun run dev
```

## Stack

React, TypeScript, Vite, Tailwind CSS
